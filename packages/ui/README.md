# @siteflow/ui

Reusable React UI components with Tailwind CSS.

## Structure

```
packages/ui/
├── src/
│   ├── components/      # UI components (Button, Input, Modal, etc.)
│   ├── hooks/           # Shared React hooks
│   ├── utils/           # UI utilities
│   └── index.ts         # Main export
├── tailwind.config.js
└── package.json
```
